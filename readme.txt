1.功能分解解析:

1.1 图片放大预览实现：
一开始的视觉效果是 将div 下的 图片部分单独导出，其实不是 只是将该图片按宽度坐标放大到 另外一个一个 div

1. 预览结构构成
原理：一共三个div   一个原图片div 一个放大镜div  一个预览div    预览div的背景图 和原图片div一样 但是图片大小得放大 ，倍数 是 原图片div  和放大镜div之间的尺寸倍数

2.预览图片跟随放大镜移动：
方法：确认好原图片div 和放大镜div 之间的倍数差距，  相应的 预览图片div背景图片 的坐标  x y都放大相应倍数
